                안녕하세요. 제 이름은 전명규라고 합니다. 저는 평소에 제가 하는 행위를 어떻게 하면 작업을 단순화시킬 수 있을지 고민을 많이 하는 편인데, 이런 성향 때문에 프로그래밍에 관심을 가지게 되었습니다. 현재는 백엔드 엔지니어지만 풀스택 엔지니어를 지향하고 있습니다.
                    
            저는 중앙대학교에서 영어영문학과 일어일문학을 전공했고 2년 동안 영어학원에서 강사로 일했습니다. 이후에 나라의 부름을 받아 공군 학사장교로 3년 3개월 동안 군복무를 수행하고, 프로그래밍에 흥미를 가지게 되어서 지금까지 이어오고 있습니다. 
          
            현재는 국비지원 it교육을 수료하고, KT 에이블 스쿨에서 빅데이터 관련 교육을 받고 있습니다. 또한 개인 사이트를 운영중입니다.
          
            제 기술 스택은 다음과 같습니다.
    • Server: AWS, Docker
    • Database: MySQL, PostgreSQL, SQLite, Redis
    • Backend: Spring, SpringBoot, Django, Express.js, Wordpress
    • Frontend: HTML/CSS, SCSS, JavaScript, Bootstrap
    • GUI: JavaFX
    • Machine Learning: Tensorflow
